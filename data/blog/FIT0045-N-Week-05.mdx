# introduction to Modules

## Modules

a **module** is a file containing python definitions and statements intended for use in other python programs
**functions** imported as part of a module live in their own **namespace**, a namespace is simply a space within which all names are distince from each other. the same name can be reused within diffrent name spaces but cannot exist within the same name space.

to import modules into a program we use the **import** function.

if you want to use a function from the imported code you have to specify with 'importedcode.function' is the general syntax for using an imported function

you could also specify if you want a specific function without refrencing the entire module with 'from importedcode import function' being the general syntax for importing a specific function into your own module

```py
#example 1
# This allows you to import the entire module
import morecode

morecode.f1() # This invokes the f1 function from the morecode module.

#example 2
# This allows you to import the f1 function from the morecode module
from morecode import f1

f1() # You are allowed to invoke the function directly

#example 3
# This allows you to import the f1 function
# from the morecode module aliased as function_1
from morecode import f1 as function_1

function_1() # You are allowed to invoke the function through the alias name

```

## decomposition

**decomposition** is the breaking down of larger comlex problems into simple smaller one's that once combined will achiece a solution

this is an example of decomposition as the problem of making a calculator has been broken down and seperated into smaller lesser problems

```py

def calculator() -> None:
    """
    Reads two numbers from the user, then ask them
    which operation they want to do on the two numbers,
    then finally prints the result.
    """
    num_1,num_2 = input_numbers()
    operation_num = input_mathematical_operation()
    if operation_num == 1:
        result = add(num_1,num_2)
    elif operation_num == 2:
        result = divide(num_1,num_2)
    elif operation_num == 3:
        result = subtract(num_1,num_2)
    print_result(result)

def input_numbers() -> (float, float):
    """
    Asks the user to input two float numbers and returns a tuple of them.
    """
    print('Enter two numbers')
    num_1 = float(input('Enter the first number:'))
    num_2 = float(input('Enter the second number:'))
    return num_1, num_2

def input_mathematical_operation() -> int:
    """
    Asks a user to input a number to select an operation, and return the number.
    The numbers associated to the operations are:
    1 <-> Add
    2 <-> Divide
    3 <-> Substract
    """
    print('Please select the mathematical operation you want to carry out')
    print('1.Add')
    print('2.Divide')
    print('3.Subtract')
    user_input = int(input())
    return user_input

def add(num_1: float, num_2: float) -> float:
    """ Returns the sum of the two numbers in parameter."""
    total = num_1 + num_2
    return total

def subtract(num_1: float, num_2: float) -> float:
    """ Returns the result of the difference between
    the first number in parameter, divided by the second."""
    difference = num_1-num_2
    return difference

def divide(num_1: float, num_2: float) -> float:
    """ Returns the result of the division of
    the first number in parameter, divided by the second."""
    quotient = num_1/num_2
    return quotient

def print_result(result: object):
    """ Prints a sentence with the result in parameter. """
    print(f'The result of your calculation is {result}')

calculator()

```
